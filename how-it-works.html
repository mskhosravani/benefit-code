How UPass Works

The UPass program provides discounted public transportation for registered students through a benefit code system.

Here’s how it works:

  • All eligible students are added to the system by the admin.
  • Each student is assigned a unique benefit code.
  • Only one benefit code can be active per student at a time.
  • Students can check their benefit code by entering their student number on the home page.
  • If a benefit code is assigned, it will be shown on the screen (and emailed later, if supported).

Using Your Benefit Code

You can use your benefit code in the official transit app or at the transit kiosk to activate your discounted travel pass.

Need Help?

If you didn’t receive a code, make sure you are registered for the current term. Contact the Student Union for support.
